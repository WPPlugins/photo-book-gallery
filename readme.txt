UNLIMITED Page Flip Books for Images with Custom Settings

== Description ==
Photo Book Gallery lets you to create Responsive Flip Books from images with full control. You can customize your Books with bunch of settings and options. Easy to use and works perfectly on mobile devices also.

== Installation ==
1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Photo Book Gallery' and install it
3. Go to Dashboard > Photo Books -> Add New and upload images as pages of book
4. Use this shortcode [photo-book-gallery id="id"] in any page
5. Now visit that page
